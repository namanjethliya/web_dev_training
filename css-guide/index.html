01

CSS Introduction

CSS (Cascading Style Sheets) controls how HTML looks. Without CSS, every page looks like a plain text document.

3 Ways to Add CSS

External (Best)
<!-- In <head> -->
<link rel="stylesheet"
      href="styles.css" />

Separate .css file. Reusable across all pages. Always use this method.

Internal
<style>
  p { color: red; }
</style>

Inside <style> in <head>. Only for that page.

Inline (Avoid)
<p style="color:red;">
  Text
</p>

Hard to maintain. Only for emails or quick tests.

Anatomy of a CSS Rule

selector {          /* → WHO to style */
  property: value;  /* → WHAT to change: HOW to change it */
}

/* Example */
h1 {
  color: #c0392b;       /* text color */
  font-size: 2.5rem;    /* text size */
  text-align: center;   /* alignment */
  margin-bottom: 24px;  /* space below */
}

The Cascade — How CSS Decides Which Rule Wins

1. Origin

Browser default styles → User styles → Author styles (your CSS)

2. Specificity

Inline > ID (#) > Class/Attribute/Pseudo-class > Element > Universal

3. Order

When specificity is equal, the LAST rule in the file wins.

!important

Overrides everything. Use very sparingly — it breaks the natural cascade.

02

Selectors

Selectors choose which HTML elements to style. Specificity determines which selector wins when rules conflict.

/* ─── BASIC ─── */
*           { }   /* Universal — every element */
p           { }   /* Element — all <p> tags */
.card       { }   /* Class — class="card" */
#hero       { }   /* ID — id="hero" (unique per page) */

/* ─── COMBINATORS ─── */
div p       { }   /* Descendant — <p> anywhere inside <div> */
div > p     { }   /* Direct child — only immediate <p> children */
h1 + p      { }   /* Adjacent sibling — <p> immediately after <h1> */
h1 ~ p      { }   /* General sibling — all <p> after <h1> */

/* ─── ATTRIBUTE ─── */
[href]               { }  /* Has the attribute */
[type="text"]        { }  /* Attribute equals value */
[href^="https"]      { }  /* Starts with */
[href$=".pdf"]       { }  /* Ends with */
[class*="btn"]       { }  /* Contains */

/* ─── PSEUDO-CLASSES ─── */
a:hover         { }  /* Mouse over */
a:focus         { }  /* Keyboard focused */
a:active        { }  /* Being clicked */
a:visited       { }  /* Already visited */
:checked        { }  /* Checked checkbox/radio */
:disabled       { }  /* Disabled input */
:required       { }  /* Required input */
:valid          { }  /* Passes HTML validation */
:invalid        { }  /* Fails HTML validation */
:placeholder-shown { } /* Input showing placeholder */
:first-child    { }  /* First child of parent */
:last-child     { }  /* Last child of parent */
:only-child     { }  /* Only child */
:nth-child(2)   { }  /* 2nd child */
:nth-child(odd) { }  /* 1st, 3rd, 5th... */
:nth-child(even){ }  /* 2nd, 4th, 6th... */
:nth-child(3n)  { }  /* Every 3rd: 3, 6, 9... */
:nth-child(3n+1){ }  /* 1, 4, 7, 10... */
:not(.active)   { }  /* NOT matching selector */
:is(h1,h2,h3)   { }  /* Matches any in list */
:where(h1,h2)   { }  /* Same as :is but 0 specificity */
:has(img)       { }  /* Contains an image (parent selector!) */
:empty          { }  /* No children */
:root           { }  /* <html> element (highest specificity) */
:target         { }  /* Element matching URL #hash */

/* ─── PSEUDO-ELEMENTS ─── */
p::before       { content: "→ "; }  /* Insert before content */
p::after        { content: " ←"; }  /* Insert after content */
p::first-line   { }  /* First rendered line */
p::first-letter { }  /* First letter (drop caps!) */
::selection     { }  /* User-selected text */
::placeholder   { }  /* Input placeholder */
::marker        { }  /* List bullet/number */

/* ─── GROUPING ─── */
h1, h2, h3, h4 { font-family: serif; } /* Apply same styles to multiple */

The Box Model

Every HTML element is a rectangular box. The box model defines the layers of space around content.

MARGIN
BORDER
PADDING
CONTENT
width × height
.box {
  /* ── CONTENT SIZE ── */
  width: 300px;
  height: 200px;
  min-width: 100px;   max-width: 600px;
  min-height: 50px;   max-height: 400px;

  /* ── PADDING — space inside, between content and border ── */
  padding: 20px;                           /* All 4 sides */
  padding: 10px 20px;                      /* top/bottom  left/right */
  padding: 10px 20px 30px 40px;           /* top  right  bottom  left (clockwise) */
  padding-top: 10px;
  padding-right: 20px;
  padding-bottom: 30px;
  padding-left: 40px;

  /* ── BORDER ── */
  border: 2px solid #333;                  /* shorthand: width style color */
  border-width: 2px;
  border-style: solid;    /* solid | dashed | dotted | double | groove | ridge | inset | outset | none */
  border-color: #333;
  border-top: 4px dashed red;             /* Individual sides */
  border-radius: 8px;                     /* Rounded corners */
  border-radius: 50%;                     /* Circle (on square element) */
  border-radius: 12px 0 12px 0;          /* TL TR BR BL */

  /* ── MARGIN — space outside, between this and other elements ── */
  margin: 20px;
  margin: 10px auto;                       /* auto = center horizontally */
  margin-top: 0;
  margin-right: auto;
  margin-bottom: 32px;
  margin-left: auto;

  /* ── OUTLINE — like border but outside the border, doesn't affect layout ── */
  outline: 2px solid blue;
  outline-offset: 4px;                    /* Gap between border and outline */

  /* ── BOX SIZING — critical! ── */
  box-sizing: content-box;   /* Default: padding + border ADDED to width */
  box-sizing: border-box;    /* Recommended: padding + border INCLUDED in width */
}

/* Apply border-box globally — highly recommended */
*, *::before, *::after {
  box-sizing: border-box;
}
Border-box vs content-box: With content-box, a 300px wide box with 20px padding becomes 340px wide. With border-box, it stays 300px. Always use box-sizing: border-box globally.

Flexbox

Flexbox lays out items in ONE direction — row or column. Perfect for nav bars, card rows, centering content.

/* ════ CONTAINER PROPERTIES ════ */
.container {
  display: flex;
  display: inline-flex;

  /* Direction — main axis */
  flex-direction: row;            /* → (default) */
  flex-direction: row-reverse;    /* ← */
  flex-direction: column;         /* ↓ */
  flex-direction: column-reverse; /* ↑ */

  /* Wrapping */
  flex-wrap: nowrap;              /* Single line (default) */
  flex-wrap: wrap;                /* Wrap to next line */
  flex-wrap: wrap-reverse;        /* Wrap upward */

  /* Shorthand: direction + wrap */
  flex-flow: row wrap;

  /* Alignment — main axis (justify = same direction as flex-direction) */
  justify-content: flex-start;    /* Pack to start (default) */
  justify-content: flex-end;      /* Pack to end */
  justify-content: center;        /* Center */
  justify-content: space-between; /* Gaps between items, none at edges */
  justify-content: space-around;  /* Equal gaps around each item */
  justify-content: space-evenly;  /* Perfectly equal gaps everywhere */

  /* Alignment — cross axis (align = perpendicular to flex-direction) */
  align-items: stretch;           /* Fill cross-axis size (default) */
  align-items: flex-start;        /* Align to cross-start */
  align-items: flex-end;          /* Align to cross-end */
  align-items: center;            /* Center on cross axis */
  align-items: baseline;          /* Align text baselines */

  /* Multi-line cross-axis alignment (only when flex-wrap: wrap) */
  align-content: flex-start;
  align-content: center;
  align-content: space-between;
  align-content: stretch;         /* (default) */

  /* Gaps */
  gap: 16px;                      /* Both row and column */
  row-gap: 12px;
  column-gap: 24px;
}

/* ════ ITEM PROPERTIES ════ */
.item {
  /* Grow — how much extra space to take */
  flex-grow: 0;    /* Don't grow (default) */
  flex-grow: 1;    /* Grow equally */
  flex-grow: 2;    /* Take twice as much as flex-grow:1 */

  /* Shrink — allow item to shrink when container is too small */
  flex-shrink: 1;  /* Can shrink (default) */
  flex-shrink: 0;  /* Never shrink */

  /* Base size — before growing/shrinking */
  flex-basis: auto;     /* Default: use width/height */
  flex-basis: 200px;
  flex-basis: 30%;
  flex-basis: 0;        /* Ignore content size, start from 0 */

  /* Shorthand: grow shrink basis */
  flex: 1;              /* = flex: 1 1 0 (grow and shrink equally) */
  flex: 0 0 200px;      /* Fixed 200px, never grow or shrink */
  flex: 1 0 auto;       /* Grow, don't shrink, auto base */
  flex: none;           /* = flex: 0 0 auto (rigid) */

  /* Override container's align-items for this item only */
  align-self: auto;
  align-self: flex-start;
  align-self: flex-end;
  align-self: center;
  align-self: stretch;
  align-self: baseline;

  /* Reorder items visually (default 0) */
  order: -1;   /* Move to front */
  order: 0;    /* Default */
  order: 1;    /* Move to back */
}

CSS Grid

Grid arranges items in TWO dimensions simultaneously — rows AND columns. Best for page layouts, dashboards, galleries.

/* ════ CONTAINER ════ */
.grid {
  display: grid;
  display: inline-grid;

  /* Define columns */
  grid-template-columns: 200px 200px 200px;          /* 3 fixed columns */
  grid-template-columns: 1fr 2fr 1fr;               /* Fractional */
  grid-template-columns: repeat(3, 1fr);            /* 3 equal columns */
  grid-template-columns: repeat(4, minmax(100px, 1fr)); /* Min 100px, stretch */
  grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));  /* Responsive! */
  grid-template-columns: repeat(auto-fit,  minmax(200px, 1fr));  /* Responsive! */
  grid-template-columns: 200px auto 200px;          /* Sidebar-main-sidebar */

  /* Define rows */
  grid-template-rows: 80px auto 60px;               /* Header, main, footer */
  grid-template-rows: repeat(3, minmax(100px, auto));
  grid-auto-rows: minmax(100px, auto);              /* Implicit rows */
  grid-auto-columns: 200px;                         /* Implicit columns */

  /* Named template areas */
  grid-template-areas:
    "header  header  header"
    "sidebar main    main"
    "footer  footer  footer";

  /* Gaps */
  gap: 16px;
  row-gap: 12px;
  column-gap: 24px;

  /* Alignment of ALL items within their cells */
  justify-items: start | end | center | stretch;    /* Horizontal */
  align-items:   start | end | center | stretch;    /* Vertical */

  /* Alignment of grid within container */
  justify-content: start | end | center | space-between | space-around | space-evenly;
  align-content:   start | end | center | space-between;
}

/* ════ ITEM PLACEMENT ════ */
.item {
  /* Column placement */
  grid-column: 1 / 3;          /* From line 1 to line 3 */
  grid-column: 1 / -1;         /* From first to last line (full width) */
  grid-column: span 2;         /* Span 2 columns from current position */
  grid-column-start: 2;
  grid-column-end: 4;

  /* Row placement */
  grid-row: 1 / 3;
  grid-row: span 2;
  grid-row-start: 1;
  grid-row-end: 3;

  /* Named area */
  grid-area: header;
  grid-area: sidebar;
  grid-area: main;
  grid-area: footer;

  /* Shorthand: row-start / col-start / row-end / col-end */
  grid-area: 1 / 1 / 2 / 4;

  /* Override container alignment for this item */
  justify-self: start | end | center | stretch;
  align-self:   start | end | center | stretch;

  /* z-axis (for overlapping) */
  z-index: 1;
}

Positioning

.box {
  /* ── STATIC (default) ── */
  position: static;
  /* Normal flow. top/right/bottom/left have NO effect. */

  /* ── RELATIVE ── */
  position: relative;
  top: 10px;     /* Move 10px DOWN from its NORMAL position */
  left: 20px;    /* Move 20px RIGHT from its normal position */
  /* Still occupies original space in the flow */
  /* Creates a positioning context for absolute children */

  /* ── ABSOLUTE ── */
  position: absolute;
  top: 0;
  right: 0;
  /* Removed from normal flow (no space reserved) */
  /* Positioned relative to nearest POSITIONED ancestor */
  /* (ancestor must be position: relative/absolute/fixed/sticky) */
  /* Falls back to <body> if no positioned ancestor found */

  /* ── FIXED ── */
  position: fixed;
  top: 0;
  left: 0;
  width: 100%;
  /* Removed from flow. Positioned relative to VIEWPORT. */
  /* Stays fixed when page scrolls (sticky nav bars, modals) */

  /* ── STICKY ── */
  position: sticky;
  top: 20px;  /* Sticks when 20px from viewport top during scroll */
  /* Hybrid: relative UNTIL it hits the threshold, then fixed */
  /* Must have a scroll container ancestor */

  /* ── Z-INDEX (stacking order) ── */
  z-index: auto;   /* Default — stacking order from source order */
  z-index: 1;      /* Higher = on top */
  z-index: 100;
  z-index: -1;     /* Behind normal content */
  /* Only works on positioned elements (not static) */
}

/* ── Classic patterns ── */

/* Badge pinned to top-right corner */
.parent { position: relative; }
.badge  { position: absolute; top: -8px; right: -8px; }

/* Full overlay */
.overlay {
  position: absolute;
  inset: 0;  /* shorthand for top:0; right:0; bottom:0; left:0; */

  from { stroke-dashoffset: 1000; }

.element {
